# springbootaimcp
Springboot + MCP Framework + Claude Desktop client + Database

Tech stack:

Springboot-4.0.3

H2 Datbase

Claude Desktop for local 

Claude Configuration

-> Create/configure claude_desktop_config.json file (Refer file under project structure) then keep it within project folder

-> Open Cladue Desktop tool -> Got to Files and select Settings -> Go to Developer options and edit config -> Select configure claude_desktop_config.json from project folder path -> close claude desktop

-> Again open Claude Desktop using above line order -> see configure claude_desktop_config.json file path 

Eg : C:\Users\bhard\AppData\Local\Packages\Claude_xyz\LocalCache\Roaming\Claude

Here, we can see claude_desktop_config.json then open and update with our claude_desktop_config.json file (We saved updated file under project folder) and see loggers folder for monitor the flow OR errors
